RFC-PATHFINDER est un site à but non-lucratif, qui permet de retrouver des fiches RFC rapidement et simplement.

RFC-PATHFINDER est copyright (c) 2018-2018 : par l'auteur Wilfried DIAS. Ce copyright s'applique a tous
les documents dans cette archive sauf exception explicitée.

Version 1.1

Contenu de l'archive :

1.  index.html : Page d'accueil de l'application (Fichier à ouvrir pour accéder au site)
2.  file.html : Page de format des fiches (Ne pas ouvrir seul)
3.  Ressources :
	- background.png : Image de fond du site
	- favicon.ico : Favicon pour les onglets
	- logo.png : Logo de RFC-PATHFINDER apparaîssant sur toutes les pages
4.  CSS :
	- style.css : Feuille de Style personnalisée
	- boostrap* : Feuilles de style Bootstrap
5.  JS :
	- javascript.js : Script du site
6.  JSON :
	- fiche.json : Fichier JSON contenant le texte des fiches RFC

/!\ ATTENTION /!\
Le site peut ne pas fonctionner sans virtual host associée
